# Boom Play

## Quick start

- Rename `.env.example` file to `.env`.

Install via your package manager:

```bash
pnpm i
```

Start dev server

```bash
pnpm dev
```

## Deployment

- Currently, this web app is deployed into vercel. Everytime the code is pushed on `dev` branch, the code will be deployed in dev and can access on `https://dev.boom-play.com/`.
- To deploy to production, merge code from `dev` to `main`, production can access on `https://boom-play.com`
